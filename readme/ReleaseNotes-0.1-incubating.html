Apache Tamaya 0.1-incubating Release Notes

Release Summary (6th August 2015)

This is the first release of Apache Tamaya. It contains basically a rather complete base set of functionality for use of Apache Tamaya in a Java SE context. Further support for Java EE is planned to be added within the next releases. Summarizing this release provides:

  • A complete SE based API for Java 7 and Java 8
  • An implementation of the API for Java 7 and Java 8.
  • Numerous extension modules that mainly depend on the Java 7 API only. So they are usable in both usage environments.
  • A set of documentation.
  • A couple of simple examples for usage and extension.

Feel free to give any kind of feedback to our team. Visit our project site http://tamaya.incubator.apache.org for further details.
